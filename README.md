# Comparative Study on Flower Pollination Algorithm and Optimal Linear Weighted Algorithm in Cognitive Radio Network

## Overview
This project presents a comparative study between the **Flower Pollination Algorithm (FPA)** and the **Optimal Linear Weighted (OLW) Algorithm** in the context of **Cognitive Radio Networks (CRNs)**.

Cognitive Radio Networks dynamically allocate spectrum resources to improve spectrum utilization, communication efficiency, and overall network performance. Optimization algorithms play a crucial role in selecting the most efficient spectrum channels.

The objective of this study is to analyze and compare the performance, efficiency, and optimization capabilities of these two algorithms in cognitive radio environments.

---

## Objectives

- Understand the working principles of Cognitive Radio Networks.
- Implement the Flower Pollination Algorithm (FPA) for spectrum optimization.
- Implement the Optimal Linear Weighted Algorithm (OLW).
- Compare both algorithms based on performance metrics.
- Evaluate the efficiency of each algorithm in dynamic spectrum allocation.

---

## Cognitive Radio Network (CRN)

A Cognitive Radio Network is an intelligent wireless communication system that dynamically adapts its transmission parameters based on environmental conditions.

Key features include:

- Detecting unused spectrum bands
- Dynamically allocating available channels
- Improving spectrum efficiency
- Reducing interference with primary users

CRNs help solve the spectrum scarcity problem by allowing secondary users to utilize unused frequency bands without affecting licensed users.

---

## Flower Pollination Algorithm (FPA)

The Flower Pollination Algorithm is a nature-inspired optimization algorithm based on the pollination behavior of flowering plants.

### Key Concepts

- Global Pollination (cross-pollination)
- Local Pollination (self-pollination)
- Levy Flight for random search
- Pollination probability switching

### Advantages

- Strong global search capability
- Faster convergence
- Efficient for nonlinear optimization problems
- Suitable for complex search spaces

---

## Optimal Linear Weighted Algorithm (OLW)

The Optimal Linear Weighted Algorithm is a decision-making and optimization method that evaluates multiple parameters and assigns weights to determine the best solution.

### Key Concepts

- Weighted parameter evaluation
- Multi-criteria decision making
- Linear optimization strategy

### Advantages

- Simple implementation
- Low computational complexity
- Efficient for structured optimization problems
- Stable optimization results
